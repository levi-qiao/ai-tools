# AI Tools Collection

AI驱动的实用工具集合，帮助提升工作效率。

## 🛠️ 工具列表

### 1. [印花税自动匹配工具](./stamp-tax-matcher)

基于Excel VBA的印花税自动匹配系统，自动识别发票税目并计算应纳税额。

**核心功能**：
- ✅ 自动匹配印花税税目
- ✅ 支持六税两费50%减免
- ✅ 自动处理红字发票
- ✅ 季度汇总报表生成

**适用场景**：财务人员处理大量发票的印花税申报

[查看详情 →](./stamp-tax-matcher)
